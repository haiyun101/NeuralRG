# NeuralRG (Modified)

This repository contains modifications and adaptations of the Neural Network Renormalization Group (NeuralRG) framework. 

## Acknowledgements
This project is built upon the original NeuralRG codebase and paper:
* **Original Authors:** Shuo-Hui Li and Lei Wang
* **Original Paper:** [Neural Network Renormalization Group (arXiv:1802.02840)](https://arxiv.org/abs/1802.02840)
* **Original Repository:** [li012589/NeuralRG](https://github.com/li012589/NeuralRG)

## Modifications
- Adapted the Hamiltonian Monte Carlo (HMC) sampling logic for specific Ising model configurations.
- Optimized the data generation pipeline (`generate_mcmc_data.py`) for execution on high-performance computing clusters.
- Added analyzing tools to study x and m distributions
